Relações Semânticas

Esta é uma ferramente para aperfeiçoar mapeamento das relações entre os dados em um Grafo RDF, de forma que se possa observar erros ou incongruências e corrigi-las.


As relações semânticas entre os dados são tão importantes ou mais que a própria classificação dos dados, pois são elas que dão sentido para a base representada.


A entrada é um grafo rdf em turtle e tem como saída a descrição de suas relações e um grafo que as representa. O projeto foi baseado no artigo Leveraging Linked Data to Discover Semantic Relations Within Data Sources
